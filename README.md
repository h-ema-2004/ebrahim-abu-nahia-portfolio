# Portfolio — Ebrahim Abu Nahia

Ready-to-deploy portfolio for GitHub Pages.

## Quick Start
1) Replace placeholders in all files:
   - YOUR_GITHUB_USERNAME
   - REPO name (default: ebrahim-abu-nahia-portfolio)
   - youremail@example.com
   - Formspree ID in contact form (optional) 

2) Add your `cv.pdf` to the project root (same folder as index.html).

3) Push to GitHub:
   - Create repo `ebrahim-abu-nahia-portfolio` on your GitHub.
   - Upload all files (or push via git).
   - Enable GitHub Pages (Settings → Pages → Deploy from branch → main).

## Structure
- index.html (Hero + Highlights)
- projects.html (Cards grid)
- contact.html (Email + optional Formspree form)
- style.css (Dark/Light theme)
- script.js (Theme toggle, copy email, auto year)
- images/ (OG cover + placeholder project images)
- robots.txt & sitemap.xml

## Notes
- All images have `loading="lazy"` and sizes.
- Accessibility: headings order, skip link, focus outlines.
- No external fonts or heavy scripts.
